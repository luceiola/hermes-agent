# 需求规格：标记词提取与词表导出（MVP）

## 1. 目标
1. 从英文阅读材料中提取被标记单词（下划线、高亮、圈画等）。
2. 生成可阅读的 Markdown 词表。
3. 生成可打印的 PDF（教学版，含原句和例句）。

## 2. 非目标
1. 不做错题入库。
2. 不做长期学习画像。
3. 不做自动训练推荐。

## 3. 架构归属
1. 本能力挂在现有学习 profile（建议 `heidou`）下。
2. 主 Agent 负责意图判定、会话状态、路由调度。
3. subagent 负责执行提取与结构化输出。
4. tool 负责模型调用、词条补全、文档渲染。

## 4. 交互规则
1. 用户发送图片后，Bot 仅回复“已收到图片 + 图片编号 + 等待指令”。
2. 无显式指令不得触发模型调用。
3. 显式指令示例：
4. `提取标记词 #img_xxx`
5. `导出词表pdf #img_xxx`
6. `提取并导出 #img_xxx`

## 5. 输入与输出
### 5.1 输入
1. 单张图片（拍照/截图）。
2. PDF 页面（单页或多页）。

### 5.2 输出
1. JSON 结构化结果。
2. Markdown 教学版词表。
3. PDF 教学版词表。

## 6. 字段定义
1. `word`：词条（建议 lemma）。
2. `phonetic_uk`：英式音标。
3. `phonetic_us`：美式音标。
4. `pos`：词性。
5. `meaning_zh`：词典式完整中文释义。
6. `simple_en_explain`：简明英文解释。
7. `example_sentence`：教学例句。
8. `source_sentence`：材料中的原句。
9. `confidence`：抽取置信度（0~1）。
10. `page`：页码（可选）。
11. `bbox`：位置坐标（可选）。

## 7. 分区规则
1. 主词表区：`confidence >= 0.75`。
2. 疑似区：`confidence < 0.75`。
3. 阈值默认 0.75，可配置。

## 8. PDF 版式（教学版）
1. 文档头：标题、时间、任务 ID、统计。
2. 主词表区：完整字段展示。
3. 疑似区：独立章节并标注“建议人工复核”。
4. 页脚：任务 ID 与生成时间。

## 9. 验收标准（MVP）
1. 发图不自动执行，必须显式命令触发。
2. 主词表 Precision 目标：>= 85%（以抽样人工校验为准）。
3. 单页处理时延目标：< 15 秒（不含排队）。
4. Markdown 可读且结构稳定，PDF 可直接打印。

## 10. 风险与约束
1. 手写标注、荧光笔覆盖可能导致 OCR/定位误差。
2. 低质量拍照会显著影响抽取质量。
3. 部分生词可能存在多词性/多释义歧义，需要上下文约束。
